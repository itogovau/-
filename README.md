# Weather Diary

Автор: [Назипов Леонид]

## Описание

Приложение "Weather Diary" позволяет вести дневник погоды с возможностью:
* Добавлять записи с датой, температурой, описанием и информацией об осадках.
* Просматривать все записи в табличном виде.
* Фильтровать записи по дате и температуре.
* Сохранять и загружать данные в формате JSON.
